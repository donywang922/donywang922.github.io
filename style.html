What idiot is so boring and insists on using pure CSS to achieve strange effects? Oh, it's me.

The code below is not complete, only the key parts. Remember that the source code of the entire website is at https://github.com/donywang922/donywang922.github.io.

A toggle button implemented in pure CSS that can also be used for form values. 
<style>
.toggle label {
    display: none;
}
.toggle input {
    display: none;
}
.toggle input:checked + label {
    display: flex;
}
</style>
<div class="toggle">
<input type="radio" id="tg2" checked name="toggle">
<label for="tg1">↑</label>
<input type="radio" id="tg1" name="toggle">
<label for="tg2">↓</label>
</div>
A pure CSS carousel, but unfortunately it cannot play automatically. 
<style>
.carousel {
    display: flex;
    justify-content: center;
    align-items: flex-end;
    position: relative;
}
.carousel .side {
    position: absolute;
    width: 100%;
    height: 100%;
    z-index: 10;
    opacity: 1;
}
.carousel input:checked ~ .side {
    opacity: 0;
}
.carousel input {
    display: none;
}
.carousel label {
    background-color: #0004;
    z-index: 11;
}
.carousel input:checked + label {
    background-color: #0008;
}
</style>
<div class="carousel">
<div class="side" style="background: #b6f688">1</div>
<input type="radio" name="carousel" id="s1" checked>
<label for="s1"></label>
<div class="side" style="background: #fde37f">2</div>

<label for="s2"></label>
